# Play Store Assets

This folder holds the Android Play Store submission working files for ToolTok.

## Contents

- `listing.md`: app title, short description, and full description draft.
- `raw-captures/`: source screenshots captured from the connected Pixel 8 Pro.

## Local release output

The current release app bundle is generated locally at:

- `app/build/outputs/bundle/release/app-release.aab`

## Notes

- The raw screenshots in `raw-captures/` are source material for the final Play Store image set.
- The Android app currently uses AdMob interstitials in release builds and Google sample test ads in debug builds.
- Privacy Policy and Terms of Service are published on GitHub Pages and linked from the site.
